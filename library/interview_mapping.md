# Interview Mapping (Network / Infra Ops / Security Analyst)

## Junior Network Engineer

Expected demonstrations:

- Read `ip addr`, `ip route`, `ip neigh` accurately
- Explain subnetting boundaries and gateway usage
- Perform a basic DNS and connectivity triage sequence
- Distinguish routing problems from service problems

Practice in this repo:

- Labs 01-03
- `docs/fundamentals/`
- `library/subnetting_quick_ref.md`

## Infrastructure Operations Analyst

Expected demonstrations:

- Apply and validate firewall/NAT changes safely
- Use packet capture for evidence-based troubleshooting
- Diagnose latency/loss vs service errors
- Document changes and rollback steps

Practice in this repo:

- Labs 04-08
- `scripts/` simulations and diagnostics
- `incidents/ticket_01_*` through `ticket_05_*`

## Security Analyst (Network-Aware)

Expected demonstrations:

- Identify exposed ports and misconfigurations
- Distinguish blocked traffic from application failures
- Triage DNS/firewall issues from logs + packet evidence
- Communicate risk and remediation clearly

Practice in this repo:

- Lab 09
- `docs/security/`
- `incidents/` ticket drills
